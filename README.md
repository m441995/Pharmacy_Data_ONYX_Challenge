# 💊 Pharmacy Data ONYX Challenge

## 📊 Project Overview

This project is part of the ONYX Data Challenge, focusing on analyzing pharmacy sales performance across multiple European regions.

## 🚀 Tools Used

* Power BI
* Data Modeling (Star Schema)
* DAX
* Data Visualization

## 📌 Key Insights

* Identified top-performing regions by revenue
* Compared urban vs rural pharmacy performance
* Analyzed total revenue, margin, and units sold
* Built interactive filters for dynamic exploration

## 🗺️ Dashboard Features

* Map visualization of pharmacy locations
* Revenue breakdown by region
* KPI table (Revenue, Units, Margin)
* Dynamic slicers (Country, Region, Pharmacy Type)
